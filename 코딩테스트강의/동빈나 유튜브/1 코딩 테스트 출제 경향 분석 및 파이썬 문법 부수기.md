### 시간제한 1초인 문제
- N의 범위가 500인 경우 : O(N³)
- N의 범위가 2,000인 경우 : O(N²)
- N의 범위가 100,000인 경우 : O(NlogN)
- N의 범위가 10,000,000인 경우 : O(N)

### 문제 해결 과정
1. 지문 읽기
2. 요구사항(복잡도) 분석
3. 아이디어 찾기
4. 소스코드 설계 및 코딩
